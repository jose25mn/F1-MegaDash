      +Para rodar o projeto localmente:
      
     1. Instale Node.js (inclui npm) e certifique-se de que o ambiente consegue instalar projetos React.
     2. Na raiz do projeto, rode:
     bash
     npm install

    Isso instalará o React e todas as demais dependências listadas no `package.json`.
    3.Após a instalação, suba o app (exemplo):
    bash
      npm run dev
    
    4.Use o script que preferir (`npm run dev:client` para o front, `npm run dev` para o servidor).
  
    5.Certifique-se de ter o banco configurado conforme as variáveis de ambiente de conexão PostgreSQL utilizadas no
         projeto.
